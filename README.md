# Taller de reportes en Quarto 

Sofía Zorrilla Azcué

**Objetivo**

Introducción para aprender a realizar reportes en Quarto desde RStudio. 

**Temas**

1. Diferencias entre Quarto y Rmarkdown
2. Proceso de creación de reportes durante la investigación 
3. Tipos de documentos que se pueden hacer en Quarto
4. Estructura de un documento de Quarto
5. Dónde buscar ayuda y más funcionalidades
6. Introducción al lenguaje de markdown
7. Código en un documento (html y pdf)
8. Imágenes
9. Tablas
10. Modificación de formato
11. Proyectos de Quarto
